## 2. Learning intent

```text
Primary concept:
Logic must be written down in shared representations so it can be understood, reviewed, and improved.

Supporting concepts:
- The same logic can be represented in more than one way
- Representations make thinking visible

Explicit non-goals:
- Writing JavaScript
- Learning full algorithm design
- Optimizing logic
- Debugging or testing logic
```

**Focus rule:** This station is about *representation*, not correctness or implementation.
